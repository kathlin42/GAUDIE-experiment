# Sound rating app

## Install with Virtual Env

Create venv

    python3 -m venv sound_rating_venv 

Activate venv
    
    Windows:
    sound_rating_venv\Scripts\activate.bat

    Unix/MacOS:
    source sound_rating_venv/bin/activate

Deacitvate venv
    
    deactivate

## Install modules
    
    pip install -r requirements.txt


## Start app 

    python3 app.py
